Section 01

🧭 Navigation Flow

Chuyện gì xảy ra khi bạn nhập URL và nhấn Enter?

⌨️
URL Input

URL Parsing & HSTS Check

Browser kiểm tra input: URL hay search query? Thêm https:// nếu thiếu. Kiểm tra HSTS list để tự động chuyển HTTP → HTTPS.

🔍
DNS Lookup

DNS Resolution

Tra cứu qua: Browser Cache → OS Cache → Router → ISP DNS → Root Server → TLD → Authoritative DNS. Kết quả: IP Address.

🤝
TCP Handshake

TCP 3-Way Handshake

SYN → SYN-ACK → ACK (~1 RTT). Thiết lập kết nối tin cậy giữa client và server.

🔐
TLS

TLS Handshake

TLS 1.3 chỉ cần 1 RTT. Trao đổi keys, xác thực certificate, mã hóa kết nối.

📡
HTTP Request

HTTP GET Request

Gửi request với headers: User-Agent, Accept, Cookie, etc. Server xử lý và chuẩn bị response.

📦
Response

HTTP Response

Server trả về status code (200 OK), headers, và HTML body. Browser kiểm tra Content-Type.

🔄
Commit

Commit Navigation

Browser Process chuyển data sang Renderer Process. IPC (Inter-Process Communication) giữa 2 processes.

🖥️
Render

Start Rendering

Renderer Process bắt đầu parse HTML, tải resources, và vẽ trang. Chi tiết xem phần Rendering Pipeline ↓

    •

  • Chỉ cần Composite trên GPU

  • Element nằm trên layer riêng

  • Luôn đạt 60 FPS
