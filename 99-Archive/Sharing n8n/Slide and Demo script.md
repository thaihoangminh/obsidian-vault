## Slide 4 – Basic Workflow Components

**Title:** What Makes Up a Workflow?  

**Key Elements:**  
- **Trigger**: Starts the workflow (e.g., form submission)  
- **Nodes**: Steps that process data (e.g., save to sheet, send message)  
- **Data Flow**: Information passed from node to node

**Script:**  
"Each workflow starts with a Trigger, like a form submission or a scheduled time. Then we add Nodes — each performing a specific task. The flow between them shows how data moves through the system."
